About This Project

This project demonstrates successful automation of data extraction from the SEBI website for portfolio managers. The data is presented in a clean, readable HTML format that makes it easy to analyze and understand the portfolio managers' information.

The automation script can extract data for any portfolio manager listed on the SEBI website, for any available month and year.
